About this scanner

A simple phishing triage app for URL, file, and message review.

Back to home

This project combines website screenshots, WHOIS lookup, VirusTotal file scanning, and Gemini-powered message analysis to help identify suspicious content quickly.

URL screenshots WHOIS metadata VirusTotal scanning Gemini AI review Search history

How it works

  • Submit a URL to inspect the site, capture a screenshot, and review WHOIS data.
  • Upload a file to VirusTotal to check for malicious detections.
  • Paste a message or email text to analyze it with Gemini and extract links.
  • Sign in to save scans in your personal history.

Privacy note

The app stores scan history only for signed-in users in the local SQLite database bundled with the project.
