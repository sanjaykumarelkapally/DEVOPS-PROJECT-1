# FlaskApp

Security scanning demo for URLs, files, and pasted messages.

## Setup

1. Create a virtual environment and install dependencies from `requirements.txt`.
2. Copy `.env.example` to `.env` and fill in the API keys.
3. Run `python app.py` from the `FlaskApp` directory.

## Features

- URL screenshot capture
- Whois lookup for scanned domains
- VirusTotal file upload and analysis
- Message scanning via URL extraction
- User sign-up and sign-in
- Personal search history for authenticated users
